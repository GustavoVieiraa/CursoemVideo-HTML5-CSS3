height = Altura.
widht = Largura.
border = Uma borda é uma linha que circula o seu conteúdo.
padding = um acochoamento ou um preenchimento, que é da borda para dentro
margin = é o espaço externo da borda, é a margem
outline = é um traçado que acompanha a borda pelo lado de fora.

Existem tipos de Caixas

  • box-level = box-level sempre se inicia em uma nova linha.
    box-level é sempre 100% área branca do seu processador.

  • inline-level = ele não vai pular para próxima linha, ele vai pular para o lado.
    e ele não quebra a linha.
